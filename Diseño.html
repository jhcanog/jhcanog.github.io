QUE ES EL DISEÑO WEB?

El diseño web es la implemetacion de conceptos artisticos que se utilizan para desarrolar un pagina web, como lo son los espacios , la simetria y el manejo de lineas , el uso de colores entre otros. Es el diseñador web quien se encarga de que la pagina web sea facil de navegar, que tenga una buena interaccion con imagenes,audio,videos en los enlaces web, es el que diseña toda la aprte visual de la web, eb resumen el diseñador se encarga de elegir que diseño tendra la pagina web:

TIPS PARA UN BUEN DISEÑO WEB

Accesibilidad y Usabilidad:

En el diseño web, la experiencia del usuario es de vital importancia. Todo debe ser pensado para darle facilidad a la hora de navegar. Elimine clics o pasos intermedios, recuerde, entre menos clics tenga que hacer el usuario para conseguir su objetivo, menor será la tasa de abandono. La información debe estar bien organizada y jerarquizada con coherencia. La tipografía debe ser legible y clara

Use Elementos Simples

No se necesita poner muchos gráficos ni llenar el sitio de colores, a veces menos es más. Lo ideal es que ofrezca una comunicación simple y amigable que sea capaz de atrapar a los usuarios. La simplicidad y limpieza en el diseño web es clave.

De le a su cliente lo que busca

Sea puntual con la información que pone, que sea precisamente lo que hace u ofrece, debe enfocar los cuerpos de texto desde un punto de vista novedoso, que los haga frescos y originales. Atrápelos con un título poderoso o una imagen de producto deslumbrante.

Crea Estructuras Mediante El DIseño

Usa gráficos que sean capaces de guiar al usuario hacia alguna acción determinada, esto se maneja mediante las características de color, tamaño y forma. Es muy importante destacar los “call to action” cuidando la línea estética. Lo ideal es lograr su visibilidad usando colores o tipografías atractivas, sin saturar el ojo del usuario.

Las imágenes son lo primero que atrae la atención de las visitas de un sitio Web. El ojo casi siempre se va automáticamente a la parte más visual e ilustrativa, es por eso que debemos estar seguros de usar imágenes de calidad, con buena resolución y que sean atractivas e interesantes para los clientes.

LENGUAJES DE PROGRAMACION

En la actualidad, existe una gran cantidad de lenguajes de programación enfocados al desarrollo web. En sus orígenes, estos lenguajes tenían un formato estático. Sin embargo, con el paso de los años y la evolución a la hora de crear páginas web han evolucionado en lenguajes dinámicos. Estos nuevos avances permiten al usuario interaccionar más con la página y mejorar la experiencia en la navegación, con lo que ya no es un mero tablón que muestra información.Como hemos dicho, existen muchos lenguajes. Para conocer un poco mejor qué tipos de lenguaje de programación web existen, es necesario hacer un recopilatorio de los más utilizados y explicar sus ventajas e inconvenientes, en comparación a los demás.

Java script

Se utiliza principalmente del lado del cliente aunque se puede utilizar del lado del servidor. Actualmente y gracias a tecnologías como AJAX es utilizado para enviar y recibir información del servidor.Como principales ventajas, tenemos que destacar que es un lenguaje de scripting seguro y fiable, cuyos scripts tienen capacidades limitadas, debido a la seguridad. Como desventajas, podríamos mencionar que el código debe descargarse por completo y es visible por cualquier usuario.

.

Php

Es un lenguaje enfocado en la creación de webs dinámicas. Sus scripts son interpretados por el servidor y genera código HTML. Requiere Apache o IIS con librerías de PHP. Hereda su sintaxis de C, Java y Perl. Como principales ventajas, hemos de decir que es un lenguaje fácil de aprender y muy rápido. Soporta la orientación a objetos y utiliza un lenguaje multiplataforma. Además, puede conectarse con una gran cantidad de base de datos: MySQL, PostgreSQL, Oracle, MS SQL Server… No necesita que se definan los tipos de variables. Uno de sus aspectos más llamativos es que está diseñado con el fin de ser un lenguaje muy seguro para escribir CGI, más que Perl o C.Es el lenguaje base que utilizan la mayoría de CMS o gestores de contenidos más extendidos como WordPress, PrestaShop, Drupal o Joomla!

Python

Considerado por muchos el lenguaje más limpio a la hora de programar. El código, al igual que JavaScript, es interpretado y no compilado. Algo curioso en este lenguaje es que permite a los programadores elegir un estilo de programación concreto (objetos, estructurado, funcional…), debido a que es un lenguaje de programación multiplataforma. Como ventajas de Python, destacamos que es libre y de fuente abierta, de propósito general. Cuenta con muchas funciones y librerías y es multiplataforma y fácil de programar. Por otro lado, su principal desventaja es que, al ser un lenguaje interpretado, es bastante lento.

COMO ELEGIR EL ENTORNO DE DESARROLLO WEB

Lo primero que debemos hacer a la hora de elegir un entorno de desarrollo o IDE (del inglés Integrated Development Environment) es tener claro los lenguajes de programación que vamos a utilizar. Como estamos hablando de desarrollo web tenemos claro que si o si vamos a utilizar HTML, CSS y JavaScript (Frontend). En la parte de servidor (Backend) tenemos varias opciones, las más comunes son PHP, Java y ASP.NET. Dependiendo del Backend elegiremos un IDE u otro.

4. Integración con un sistema de control de versiones.

7. Ejecución en modo debug.

8. Buscar y remplazar código.

9. Refactorizar código.
